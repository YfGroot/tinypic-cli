# tinypic-cli
## install
  ```bash
    npm i -g tinypic-cli
  ```
## how to use
```text
  Usage
    tiny <file or path>
    tiny -b   // backup all your images into \`_folder\`

  Example
    tiny      // current dir
    tiny .    // current dir
    tiny a.jpg
    tiny a.jpg b.jpg
    tiny img/test.jpg
    tiny folder
```
