# tinypic-cli
## install
  ```bash
    npm i -g tinypic-cli
  ```
## how to use
```text
  Usage
    tiny <file or path>

  Example
    tiny
    tiny .
    tiny a.jpg
    tiny a.jpg b.jpg
    tiny img/test.jpg
```
